# Школьный Telegram-бот

В проекте собран внутренний школьный мессенджер на `Python` и `pyTelegramBotAPI` с регистрацией, админ-разделом, диалогами и записью в прачечную.

## Что умеет бот

- регистрация с ролями и подтверждением администратора;
- отмена регистрации и повторная подача заявки;
- отправка текста, фото, видео и файлов;
- входящие уведомления и список диалогов;
- ответы на сообщения;
- админ-раздел со статистикой, заявками и историей прачечной;
- запись в прачечную через `/signup`;
- логирование действий в отдельный файл.

## Основные команды

- `/register` — создать или обновить заявку;
- `/cancel_registration` — отменить текущую регистрацию;
- `/send` — отправить сообщение;
- `/read` — открыть список уведомлений;
- `/dialogs` — открыть диалоги;
- `/signup` — запись в прачечную;
- `/admin` — админ-раздел;
- `/pending` — быстрый доступ к заявкам.

## Установка

1. Установите зависимости:

```powershell
py -m pip install -r requirements.txt
```

2. Создайте `.env`:

```powershell
Copy-Item .env.example .env
```

3. Заполните `.env`:

```env
BOT_TOKEN=ваш_токен_бота
DATABASE_PATH=school_messenger.db
BOT_ADMIN_IDS=123456789,987654321
PAGE_SIZE=6
ACTIVITY_LOG_PATH=logs/bot_activity.log
```

4. Запустите бота:

```powershell
py bot.py
```

## Прачечная

- запись доступна на ближайшие 2 дня;
- время округляется вверх до ближайших `:00` или `:30`;
- окна доступны с `07:00` до `22:30` по Москве;
- всего `3` стиралки и `2` сушки;
- пользователь может держать максимум одну будущую запись на стирку и одну будущую запись на сушку;
- для стирки доступны режимы `Смешанные вещи` (`1.5 часа`) и `Хлопок` (`3 часа`);
- для сушки используется длительность `3 часа`;
- будущую запись можно отменить.

## Журнал действий

Файл журнала задается через `ACTIVITY_LOG_PATH`. В него записываются:

- отправленные сообщения;
- регистрация, одобрение, отклонение и отмена регистрации;
- создание и отмена записей в прачечную.
